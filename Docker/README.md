### Docker 命令

| 命令          | 用途                              |
| ------------- | --------------------------------- |
| docker pull   | 获取 iamge                        |
| docker  build | 创建 iamge                        |
| docker images | 列出 iamges                       |
| docker run    | 运行 container                    |
| docker ps     | 列出 container                    |
| docker rm     | 删除 container                    |
| docker rmi    | 删除 image                        |
| docker cp     | 在 host 和 container 之间拷贝文件 |
| docker commit | 保存改动为新的 image              |
